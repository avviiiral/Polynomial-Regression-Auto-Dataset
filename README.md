# Polynomial Regression on Auto Dataset

## Project Overview
This project focuses on implementing **Polynomial Regression** using Python to model non-linear relationships in data.  
The Auto dataset is used to demonstrate how polynomial features help improve predictions when linear regression is not sufficient.

This project is beginner-friendly and useful for understanding regression concepts in machine learning.

---

## Objective
- Understand the limitation of linear regression
- Apply polynomial regression for non-linear data
- Visualize and analyze model behavior with different polynomial degrees

---

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

---

## Dataset
The project uses a cleaned version of the Auto dataset (`auto_islr_cleaned.csv`) for regression analysis.

---

## Methodology
- Load and explore the dataset
- Preprocess the data
- Generate polynomial features
- Train polynomial regression models
- Visualize regression curves
- Analyze model performance

---

## Results
- Polynomial regression captures non-linear patterns better than linear regression
- Model complexity increases with polynomial degree
- Visual plots help understand overfitting and underfitting

---

## How to Run
1. Clone the repository  
2. Open the Jupyter Notebook  
3. Run all cells in `polynomila regretion.ipynb`
